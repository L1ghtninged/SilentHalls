# Silent Halls

## 3D RPG Dungeon Crawler in Unity

**Silent Halls** is a 3D dungeon crawler RPG developed in the Unity game engine. The project focuses on the design and implementation of gameplay mechanics in a three-dimensional environment, with emphasis on grid-based movement, dungeon exploration, and a combat system.

---

## Project Abstract

This work focuses on the design and implementation of a 3D RPG dungeon crawler game created in the Unity game engine. The topic is centered around video game development and the implementation of gameplay mechanics in a three-dimensional environment. The goal was to design a fully functional game in which the player moves on a grid in 3D space, explores game levels, fights enemies, and solves logical puzzles.

The game system also includes character progression through a skill tree, allowing gradual improvement of the player’s abilities. The project was implemented using the C# programming language and the Unity engine. The result is a fully playable 3D game that fulfills the defined objectives.

The contribution of this project lies in the practical application of knowledge in object-oriented programming and game system design.

---

## Game Concept

The game combines elements of classic dungeon crawlers with modern 3D implementation:

- Grid-based movement in a 3D environment  
- Exploration of dark dungeons  
- Combat system with enemies  
- Logical puzzles and environmental interaction  
- Character progression through a skill tree
